# Ticket 001 - Create Azure Resource Group

## Business Objective

Deploy the first Azure resource for the Enterprise Azure Cloud Homelab by creating a Resource Group to organize and manage all project resources.

---

## Resource Created

**Resource Type:** Resource Group

**Name:** rg-enterprise-cloud-lab-dev

**Region:** East US 2

**Subscription:** Azure for Students

---

## Why We Created It

A Resource Group is a logical container that holds related Azure resources.

It allows us to:
- Organize project resources
- Manage resources together
- Track costs
- Apply permissions (RBAC)
- Delete the entire environment when it's no longer needed

---

## What I Learned

Today I learned that a Resource Group is the foundation of an Azure project. Every resource in this homelab will be placed inside this Resource Group so it can be managed as a single environment.
